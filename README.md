## 🛠️ 实用软件推荐

### GitHub Desktop

- 简介：GitHub 官方 Git 图形化工具，简单易用，适合新手管理代码仓库
- 适用：版本管理、项目提交、协作开发
- 下载：[https://desktop.github.com/](https://desktop.github.com/)

### VS Code

- 简介：轻量强大的代码编辑器，支持几乎所有语言
- 下载：[https://code.visualstudio.com/](https://code.visualstudio.com/)

### Obsidian

- 简介：本地优先的知识管理工具，适合做笔记、知识库、学习总结
- 下载：[https://obsidian.md/](https://obsidian.md/)

---

## 🧰 常用 VS Code 插件

- **Chinese (Simplified)**：汉化界面
- **Prettier**：代码自动格式化
- **Auto Rename Tag**：自动重命名配对标签
- **Markdown All in One**：Markdown 写作增强
- **GitLens**：Git 增强工具，查看提交记录
